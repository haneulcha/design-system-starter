## Reading the Origin Share
_Each role's hex values are classified against that system's accent profile + a chroma threshold. neutral = c ≤ 0.025; accent = c ≥ 0.05 AND hue within ±20° of the system's primary accent; unique = chromatic but not the primary accent (e.g. semantic green/red); no-color = row had no parseable hex._
